# Google Apps Rating
![Python 3.6](https://img.shields.io/badge/Python-3.6-brightgreen.svg) ![matplotlib](https://img.shields.io/badge/Library-Matplotlib-orange.svg)
This repository contains CSV taken from Kaggle.

## Problem Statement
As we all know, Apps are used world wide with rapid increase of users Now, given the dataset of google playstore apps, define ratings, coments, price, usage, etc.

## Steps
- Read data
- Data cleaning
- Rating Outliers >5
- Data Imputation and Manipulation
- Data Visualization

## Algorithm Used
- LogisticRegression
- AdaBoostClassifier
